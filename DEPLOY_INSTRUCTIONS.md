## Step 2: Connect Netlify
1. Go to [netlify.com](https://netlify.com)
2. "Add new site" → "Import existing project"  
3. Connect to GitHub repository
4. Build command: `npm run build`
5. Publish directory: `.next`
6. Deploy!
